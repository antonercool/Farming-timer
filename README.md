# Farming-timer
This is a simple timer for farming in maplestory

## Easy way ##
- pull the repo
- Run the .exe in /dist


### Prereqs ###
- install python3
- install pip

### Libs ###
- pip install pygame 

### how to run ###
default: 90 farming - 30 looting
- ./farmer.py                    

custom:
- ./farmer.py 90 20
